# Phantom Sim Telemetry — инструкция

Живая телеметрия iRacing из машины пилота в тот же интерфейс у стратега, по сети
Radmin. Бесплатно, без McLaren ATLAS и без лицензий.

```
  iRacing (пилот, за рулём)
        │  shared memory ~60 Гц
        ▼
  Phantom Sim Telemetry ── раздаёт по Radmin (TCP :8100) ──►  то же приложение у стратега
   (видит сам, 2 вкладки)                                      (видит ВСЁ вживую)
```

Четыре worksheet-вкладки (снизу, как в MoTeC i2):
- **ГРАФИК** — каналы с курсором и маркерами кругов (L2, L3…), приборы, G-G метр,
  шины, карта трассы **с раскраской по скорости** (синий=медленно → красный=быстро),
  таблица DATA.
- **КРУГИ / ДЕЛЬТА** — анализ кругов как в MoTeC i2: наложение кругов по дистанции,
  Δ-время к эталону (лучший круг ★, белый), текущий круг рисуется live (голубой),
  галочками выбираешь какие круги наложить, курсор мышью — значения в любой точке трассы.
- **ГИСТОГРАММЫ** — распределение времени по газу / тормозу / скорости / передачам.
- **ВСЕ КАНАЛЫ** — полный список телеметрии iRacing (все ~280+ переменных) с поиском.

Переключение: F2–F5 или вкладки снизу. Пауза экрана — Ctrl+P, скриншот — Ctrl+S.

## ЧАСТЬ 1 — ПИЛОТ (за рулём, хост)

**Нужно:** iRacing, приложение Phantom Sim Telemetry, поднятый Radmin VPN.
Твой адрес в Radmin: смотри в окне **Radmin VPN** (вид `26.x.x.x`) и дай его стратегу.

1. Запусти приложение: `run.bat` (из исходников) или `dist\LiveTelemetry.exe`.
2. На стартовом экране выбери карточку **iRacing**.
3. Галочки: **«Локально»** ✅ и **«Раздавать стратегу по Radmin»** ✅. Порт **8100**.
4. Жми **ПОДКЛЮЧИТЬСЯ ▶**.
5. Если выскочит **Брандмауэр Windows** — нажми **Разрешить доступ** (частные сети).
   Не появилось — запусти `firewall-8100.cmd` (подтверди UAC). Делается один раз.
6. Выезжай на трассу — телеметрия пойдёт у тебя и автоматически уйдёт стратегу.
   Открой вкладку **ВСЕ КАНАЛЫ** — там весь список iRacing с поиском.

> Можно не раздавать (сними вторую галку) — тогда просто смотришь сам.

## ЧАСТЬ 2 — СТРАТЕГ (другой ПК)

**Нужно:** файл **`LiveTelemetry.exe`** (пришлёт пилот) и поднятый **Radmin VPN**
в той же сети. Python и iRacing стратегу НЕ нужны.

1. Подключись в **Radmin VPN** к сети пилота.
2. Запусти **`LiveTelemetry.exe`** (Windows может спросить «всё равно запустить» —
   это норм, файл неподписанный).
3. Выбери карточку **iRacing**.
4. **Сними** галочку **«Локально»**.
5. Впиши **IP-АДРЕС ХОСТА (RADMIN)** пилота и **ПОРТ** `8100`.
6. Жми **ПОДКЛЮЧИТЬСЯ ▶**.
7. Когда пилот на трассе — видишь его телеметрию тик-в-тик: дашборд и вкладку
   **ВСЕ КАНАЛЫ** с полным списком.

## F1 25 / 26 (то же самое, через UDP игры)

**Пилот (за рулём):**
1. В игре: **Settings → Telemetry → UDP Telemetry = On**, IP = `127.0.0.1`,
   Port = `20777`, Send Rate = `60 Hz`, Format = `2025` или `2026`,
   Your Telemetry = `Public`.
2. В приложении: карточка **F1 25/26** → **«Локально»** ✅ + **«Раздавать стратегу»** ✅
   → ПОДКЛЮЧИТЬСЯ. (Год болида — 25 или 26 — определяется автоматически.)

**Стратег:** карточка **F1 25/26** → снять **«Локально»** → вписать **Radmin-IP пилота**
и **порт связи 8100** → ПОДКЛЮЧИТЬСЯ.

> IP получателя для F1 вписывается **в игре** (поле UDP IP Address), а не в приложении:
> на одном ПК — `127.0.0.1`, приложение само раздаёт стратегу по мосту 8100.

## LMU (Le Mans Ultimate) — порт 8000, а НЕ 8100!

Для LMU стратег подключается не к приложению пилота, а к **серверу LMU Pit Wall**
на ПК пилота.

**Пилот (за рулём):**
1. Запусти LMU Pit Wall server (`lmu-pitwall\start-pitwall.bat`) — он слушает порт **8000**.
2. Запусти LMU (плагин rF2 Shared Memory уже стоит).
3. В приложении: карточка **LMU** → галочка **«Локально»** ✅ → ПОДКЛЮЧИТЬСЯ.

**Стратег:**
1. Карточка **LMU** → снять «Локально» → **IP пилота (Radmin)** и порт **8000** → ПОДКЛЮЧИТЬСЯ.

**Быстрая проверка сети (стратег, без приложения):** открой в браузере
`http://<Radmin-IP пилота>:8000/healthz` — если видишь JSON `{"ok":true,…}`,
сеть и сервер работают; если страница не открывается — проблема в Radmin, а не
в приложении.

## Если не видно данных
- **У стратега пусто:** пилот должен быть **на трассе**, и у него включена «Раздавать».
- **Не подключается:** Radmin поднят с обеих сторон, оба ПК в сети **зелёные**
  (жёлтый/красный = туннель не встал — перезайти в сеть, выключить другие VPN/прокси).
- **Порт:** iRacing и F1 — **8100**; **LMU — 8000** (сервер Pit Wall).
- **Роли не перепутаны:** пилот = «Локально» ✅ (+ «Раздавать» для iRacing/F1);
  IP вписывает только **стратег**, и это IP **пилота**.
- **Блокируется:** на ПК пилота при первом запуске Windows спросит про брандмауэр —
  жать **Разрешить** (обе сети). Если не спросил — `firewall-8100.cmd`.
- **Windows переименовал файл:** если скачал повторно и получил `LiveTelemetry (2).exe`
  — переименуй в `LiveTelemetry.exe`, иначе старое разрешение брандмауэра не действует.
- **IP сменился:** Radmin иногда выдаёт новый адрес — пилот сообщает актуальный.
